# Portafolio Web – Sprint 4

## Descripción del proyecto

Este proyecto es una página de portafolio personal desarrollada como parte del Sprint 4.  
La página presenta información sobre una desarrolladora Full-Stack Junior, incluyendo sus habilidades, proyectos destacados y datos de contacto.

El sitio está diseñado para ser responsivo y adaptarse a diferentes tamaños de pantalla (1440px, 768px y 320px).

## Funcionalidad

El sitio incluye:

- Un **header** con información personal y botones de acción.
- Una sección de **habilidades y herramientas** con iconos de tecnologías.
- Una sección de **proyectos destacados** con tarjetas que muestran proyectos realizados.
- Un **footer** con información de contacto y enlaces a redes sociales.
- Efectos **hover** en botones, enlaces y redes sociales.
- Diseño **responsive** para diferentes tamaños de pantalla.

## Tecnologías y técnicas utilizadas

En este proyecto se utilizaron:

- **HTML5** para la estructura semántica del sitio.
- **CSS3** para estilos y diseño visual.
- **Flexbox** para la distribución de elementos.
- **CSS Grid** para la sección de proyectos.
- **Media Queries** para el diseño responsivo.
- **Git y GitHub** para control de versiones.
- **Figma** como referencia de diseño.
